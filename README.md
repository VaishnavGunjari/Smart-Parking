# 🚗 Smart Parking System

A simple and efficient **Smart Parking System** built using Python and Flask.  
This project demonstrates how parking slots can be monitored and visualized through a web interface.

---

## 📌 Overview

The Smart Parking System helps in:

- Detecting available and occupied parking slots
- Displaying parking slot status visually
- Managing parking space efficiently
- Demonstrating a basic smart parking concept

This project can be extended further with IoT sensors, database storage, or real-time updates.

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Visualization:** Video/Image-based slot detection
- **Environment:** Localhost Web Application
